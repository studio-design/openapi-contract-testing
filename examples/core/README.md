# Core quickstart

> **Pre-release evaluation only.** Run this beta in an isolated branch or CI
> job; Gesso 2 has not reached a stable release.

```bash
composer require --dev "studio-design/gesso:^2.0@beta"
composer install
composer test
```

The passing test validates a response through the framework-independent API. The PHPUnit extension prints endpoint coverage when the suite finishes.
